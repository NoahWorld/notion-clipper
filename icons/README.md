# Icons

Please add the following icon files:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

You can create icons using:
- [Figma](https://figma.com)
- [Canva](https://canva.com)
- [IconKitchen](https://icon.kitchen/)

## Design Suggestions

- Use a simple, recognizable symbol (e.g., bookmark, clip, or "N" for Notion)
- Keep it minimal and clean
- Use colors that match Notion's brand (black/white or subtle colors)
- Ensure it's visible on both light and dark backgrounds
